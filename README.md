# Facebook

## Consignes

À partir d'un tableau `$users` contenant une série de prénoms (Simon, Arthur, Yohan, etc.), afficher selon le nombre de prénoms (0, 1, 2 ou > 2), les phrases correspondantes : 

* *"Soyez le premier à aimer cette publication"*,
* *"Simon aime cette publication"*,
* *"Simon et Arthur aiment cette publication"*,
* *"Simon et X autres personnes ont aimé cette publication."*,

> Petit bonus : afficher un prénom aléatoirement parmi ceux du tableau.

## Concepts

* Tests successifs,
* Fonctions,
* Manipulation (simple) de tableaux
